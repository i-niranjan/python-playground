# Python Revision Round 1

Use this folder as a hands-on revision track.

Suggested loop for each chapter:

1. Write your own notes in `notes.md` before looking back at old files.
2. Solve `drills.py` first. Keep these simple and fast.
3. Solve `challenges.py` next. These mix the chapter with older topics.
4. Build the `mini_project.md` task without looking at solutions.
5. After finishing, compare with your old chapter folder and update notes.

Rules:

- Do not copy from the tutorial.
- If stuck for more than 20 minutes, write down the exact confusion in notes.
- Prefer small functions over one large script.
- Run your files often.
- Keep old attempts. They show progress.
