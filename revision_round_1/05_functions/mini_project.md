# Mini Project: Calculator With Functions

Build a calculator using functions.

Functions to create:

- add(a, b)
- subtract(a, b)
- multiply(a, b)
- divide(a, b)

Then ask the user:

- first number
- second number
- operation

Call the correct function and print the result.

Extra:

- Handle division by zero.
- Keep asking until the user chooses to quit.
