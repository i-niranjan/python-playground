# Mini Project: Library System

Build a small library system using classes.

Suggested classes:

- `Book`
- `Member`
- `Library`

Book:

- title
- author
- is_borrowed

Member:

- name
- borrowed_books

Library:

- books
- members
- add_book()
- register_member()
- borrow_book()
- return_book()
- list_available_books()

Keep it in one file first. Split into modules later as extra practice.
